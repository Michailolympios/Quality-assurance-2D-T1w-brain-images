Comparison of a reference 2D binary segmentation masks by plane and tissue with two binary segmentation algorithms (you can increase or decrease the num of algorithms) in respect to accuracy, precision, true positives, true negatives, false positives false negatives and the creation of a contigency table, csv files including per image and per tissue + plane metrics. A QA reference map is also plotted with TP,TN,FP,FN with different colours per image.

You require specific libraries and set your directories accordingly based on your preference: 
Python 3.9+ 
numpy
pandas
imageio
matplotlib
